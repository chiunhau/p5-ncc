## Basic Setup ##
```js
function() {
	createCanvas(800, 600);
	frameRate(30);
}
```

## Must Know Tricks ##

#### ellipse(x, y, width, height)
畫橢圓用。
#### fill(color(r, g, b))
更改幾何圖形填充色。
#### stroke(color(r, g, b))
更改幾何圖形邊框色。
#### background(color(r, g, b))
用顏色填滿整個畫面。
#### noFill() 
不填色。
#### noStroke() 
不設邊框。
####translate(dx, dy) 
用於位移原點，即直接將座標做線性變換，方便好用。而且隨時可以切換回去。
